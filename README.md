# Connect Four | Java

This is a Connect Four client based game that was developed only with java.

## Rules Of The Game

Get 4 spaces in a colum, row or diagonal line.

## What was used?

Only Java.

### Prerequisites

Required:
- Java Developer SDK

Optional:
- Editor ([VSCode](https://code.visualstudio.com/) is recommended)
